System Abstract

Team 30 has engineered a comprehensive backend infrastructure simulating enterprise-grade e-commerce logic. The kernel features atomic inventory deduction, role-based access control (RBAC) via JWT, and a volatile in-memory database for high-speed transaction prototyping. Architectural integrity is maintained through strict TypeScript interfaces and a modular middleware stack.

PowerShell Interface

Direct command injection via Invoke-RestMethod.

$baseUrl = "http://localhost:3000/api"

1. Public Access Protocol

Open 
# 1. System Status Check
Invoke-RestMethod -Method Get -Uri "http://localhost:3000/"

# 2. Retrieve Catalog
Invoke-RestMethod -Method Get -Uri "$baseUrl/products"

# 3. Query Specific Item
Invoke-RestMethod -Method Get -Uri "$baseUrl/products/p1"

# 4. Initiate User Registration
$body = @{ email = "test@example.com"; password = "password123"; name = "Test User" } | ConvertTo-Json
Invoke-RestMethod -Method Post -Uri "$baseUrl/auth/register" -ContentType "application/json" -Body $body

2. Secure Client Protocol

Encrypted 
# 5. Authenticate & Cache Token [REQUIRED]
$loginBody = @{ email = "test@example.com"; password = "password123" } | ConvertTo-Json
$response = Invoke-RestMethod -Method Post -Uri "$baseUrl/auth/login" -ContentType "application/json" -Body $loginBody
$token = $response.token
$headers = @{ Authorization = "Bearer $token" }
Write-Host ">> ACCESS GRANTED. TOKEN CACHED."

# 6. Push to Cart
$cartItem = @{ productId = "p1"; quantity = 1 } | ConvertTo-Json
Invoke-RestMethod -Method Post -Uri "$baseUrl/cart" -Headers $headers -ContentType "application/json" -Body $cartItem

# 7. Execute Checkout
Invoke-RestMethod -Method Post -Uri "$baseUrl/orders/checkout" -Headers $headers

3. Root Access Protocol

Restricted 
# 10. Root Login
$adminData = @{ email = "admin@store.com"; password = "admin123" } | ConvertTo-Json
$res = Invoke-RestMethod -Method Post -Uri "$baseUrl/auth/login" -ContentType "application/json" -Body $adminData
$adminHeaders = @{ Authorization = "Bearer $($res.token)" }

# 11. Inject New Asset
$newProd = @{ name = "Super GPU"; price = 999; stock = 5; category = "Tech" } | ConvertTo-Json
Invoke-RestMethod -Method Post -Uri "$baseUrl/products" -Headers $adminHeaders -ContentType "application/json" -Body $newProd

# 13. Purge Asset
Invoke-RestMethod -Method Delete -Uri "$baseUrl/products/p3" -Headers $adminHeaders
